Project 1

I developed a secret number game using HTML, CSS, and JavaScript. In this game, users have 5 attempts to guess a randomly generated secret number within a given range. The game provides feedback after each guess, indicating whether the guess is too high, too low, or correct. The interface is built using HTML for structure, CSS for styling, and JavaScript for the game logic, including tracking the number of attempts and dynamically updating the user interface.
